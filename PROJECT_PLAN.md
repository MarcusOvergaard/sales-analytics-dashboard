# Sales Analytics Dashboard

## Project Plan

### Project Goal

Build an end-to-end Business Intelligence project demonstrating:

* SQL
* Data Modeling
* Data Cleaning
* Business Analytics
* Power BI
* Dashboard Design
* Business Communication

The project should simulate the workflow of a real Junior Data Analyst.

Workflow:

Raw Data → SQL Database → Analysis → Power BI Dashboard → Business Insights
